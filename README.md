# page-router
* 배포 링크입니다.
* SSR , SSG , ISR을 사용하여 구현했습니다.
* 메인페이지(ISR): 업데이트 버튼을 누를 때 재생성 하면 되므로 ISR렌더링을 이용
* 검색페이지(CSR): 쿼리스트링을 이용해서 조회를 해야하므로 클라이언트 측에서 조회 처리
* 상세페이지(SSG) : 폴백옵션을 설정해 SSG방식으로 처리
